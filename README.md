# Portfólio Acessível - Raissa Pedroso

Este projeto é um portfólio desenvolvido para a disciplina de **Acessibilidade e Inclusão Digital**, focado em garantir que a web seja um espaço para todos.

## Percepção
- **Descrições de Imagem:** Todas as imagens possuem o atributo `alt` detalhado.
- **Contraste:** Cores selecionadas para garantir a leitura por pessoas com baixa visão.
- **VLibras:** Integração com o widget de tradução para LIBRAS.

## Operação
- **Navegação por Teclado:** Uso de `skip-link` para pular o menu e foco visível em todos os elementos interativos.
- **Atalhos:** O menu mobile pode ser fechado utilizando a tecla `ESC`.
- **Sem Dependência de Mouse:** Todos os links e botões são acessíveis via `Tab`.

## Compreensão
- **Linguagem Simples:** Documentação e textos objetivos.
- **Feedback Visual:** Indicações claras de estados de erro ou sucesso (ex: botões de áudio).

## Robustez
- **HTML5 Semântico:** Uso de `<header>`, `<main>`, `<nav>` e `<section>`.
- **ARIA States:** Uso de `aria-expanded` e `aria-controls` para estados de menu dinâmicos.

## Como usar
1. Clone o repositório: `git clone https://github.com/seu-usuario/seu-repositorio.git`
2. Abra o arquivo `index.html` em seu navegador.
3. Use a tecla `Tab` para testar a navegação sem o mouse.
